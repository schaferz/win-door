# Áttekintés

Ajtó és ablak gyártást és forgalmazást támogató ERP vállalat irányítási rendszer.

# Technológia

- Angular 21.0.0
- RxJS 7.8.0
- PrimeNG 21.0.2
- Primeicons 7.0.0
- Tailwind CSS 4

# Főbb funkciók

- Ablak vagy ajtó ábrázolása bemenő paraméterek alapján (például szélesség, magasság, tok vastagság stb...)

# Felépítés

- [core](src/core): globális források
- [shared](src/shared): szélesebb körben használt, megosztott szolgáltatások, komponensek
- [features](src/features): specifikus funkciók, almappánként az adott funkció (például projektek, anyag karbantartók

# Konvenciók
